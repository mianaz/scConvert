# Write required h5seurat root attributes and empty groups

Write required h5seurat root attributes and empty groups

## Usage

``` r
.h5seurat_write_scaffold(h5, assay, n_cells)
```

## Arguments

- h5:

  H5File handle (mode = "w")

- assay:

  Character active assay name

- n_cells:

  Integer number of cells (for active.ident)
