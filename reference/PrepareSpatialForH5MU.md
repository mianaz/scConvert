# Prepare spatial data for h5mu export

Prepare spatial data for h5mu export

## Usage

``` r
PrepareSpatialForH5MU(object, assays, modality.names, verbose = FALSE)
```

## Arguments

- object:

  Seurat object

- assays:

  Assay names

- modality.names:

  Modality name mapping

- verbose:

  Logical

## Value

Modified Seurat object with spatial metadata prepared
