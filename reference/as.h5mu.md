# Save a Seurat object to h5mu format

Save a Seurat object to h5mu format

## Usage

``` r
as.h5mu(object, filename, ...)
```

## Arguments

- object:

  A Seurat object

- filename:

  Output filename

- ...:

  Additional arguments passed to
  [`writeH5MU`](https://mianaz.github.io/scConvert/reference/writeH5MU.md)

## Value

Invisibly returns the filename
