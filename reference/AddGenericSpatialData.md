# Add generic spatial data to Seurat object

Add generic spatial data to Seurat object

## Usage

``` r
AddGenericSpatialData(
  seurat_obj,
  coords,
  assay_name = "Spatial",
  verbose = TRUE
)
```

## Arguments

- seurat_obj:

  Seurat object

- coords:

  Spatial coordinates

- assay_name:

  Assay name

- verbose:

  Print messages

## Value

Modified Seurat object
