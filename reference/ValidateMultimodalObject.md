# Validate Seurat object for multimodal export

Validate Seurat object for multimodal export

## Usage

``` r
ValidateMultimodalObject(object, assays, verbose = FALSE)
```

## Arguments

- object:

  Seurat object

- assays:

  Character vector of assays to validate

- verbose:

  Logical

## Value

List with 'valid' (logical) and 'message' (character) elements
