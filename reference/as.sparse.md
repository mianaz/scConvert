# Convert an HDF5 dataset to a sparse matrix

Convert an HDF5 dataset to a sparse matrix

## Usage

``` r
# S3 method for class 'H5D'
as.sparse(x, ...)

# S3 method for class 'H5Group'
as.sparse(x, ...)
```

## Arguments

- x:

  An HDF5 dataset or group

- ...:

  Additional arguments (ignored)

## Value

A sparse matrix
