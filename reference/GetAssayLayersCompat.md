# Get available layers or slots from assay

Returns list of available data layers (V5) or slots (V4) from an Assay
or Assay5 object.

## Usage

``` r
GetAssayLayersCompat(object, verbose = FALSE)
```

## Arguments

- object:

  Assay or Assay5 object

- verbose:

  Print diagnostic messages

## Value

Character vector of layer/slot names
