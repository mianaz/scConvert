# Convert compound dataset to group

Convert compound dataset to group

## Usage

``` r
CompoundToGroup(src, dst, dname, order, index = NULL, name_map_fn = NULL)
```

## Arguments

- src:

  Source compound dataset

- dst:

  Destination file

- dname:

  Destination name

- order:

  Order attribute name

- index:

  Index for subsetting

- name_map_fn:

  Optional function to map column names
